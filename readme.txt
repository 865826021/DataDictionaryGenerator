一.功能说明
1.数据字典目前支持mysql,sql server。
2.生成的数据字典带文档结构图
3.同时支持mysql的网页版跳转备份，也支持计划任务下一键式备份

二.使用方法
1.打开inc/site.inc.php 修改你本地配置的访问路径
2.在当前目录下创建tmp目录，有写权限
3.运行default.php开始配置
